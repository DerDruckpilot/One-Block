# Entscheidung 0001 – Plattform: Web-App/PWA

## Entscheidung

One Block wird als statische Web-App/PWA umgesetzt und über GitHub Pages gehostet.

## Gründe

- läuft auf iPhone und iPad
- kein App-Store-Zwang für frühe Tests
- einfache Verteilung
- Codex kann gut im Repo arbeiten
- keine Serverinfrastruktur nötig

## Konsequenz

Die Technik bleibt zunächst bei HTML, CSS, JavaScript-Modulen und Canvas 2D.
