# Entscheidung 0002 – Pixelgrößen

## Entscheidung

- Welt-Tiles: 32x32 Pixel
- Spieler-Ingame-Sprite: 48x48 Pixel pro Frame

## Gründe

- gute Lesbarkeit auf iPhone/iPad
- sauberer Pixelart-Look
- genug Platz für Charakterdetails
- einfache Tilemap-Logik

## Konsequenz

Alle Runtime-Assets müssen passend zu diesen Größen erstellt oder sauber darauf zugeschnitten werden.
