# Entscheidung 0004 – Weltkante und Void

## Entscheidung

Bodengebundene Tiere und Gegner laufen nicht freiwillig über die Weltkante. Treffer und Knockback können sie aber über den Rand stoßen.

## Loot

Loot erscheint auf dem letzten gültigen Tile oder direkt an der Kante.

## Fliegende Gegner

Fliegende Gegner fallen nicht herunter. Sie dürfen über den Void fliegen, bleiben aber maximal 3 Blöcke vom nächsten bebauten Tile entfernt.
