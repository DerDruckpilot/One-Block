# Assets

Alle finalen Spielgrafiken sollen als PNG im passenden Pixelraster abgelegt werden.

Wichtige Größen:

- Welt-Tiles: 32x32 px
- Spieler-Ingame-Sprite: 48x48 px pro Frame
- kleine Tiere: 32x32 px
- mittlere Tiere/Gegner: 48x48 px
- große Tiere: 64x64 px

Die vorhandenen PNGs sind nur Platzhalter, aber im korrekten Raster angelegt.
